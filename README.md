# 🏥 Healthcare Appointment Management System

A web application to manage healthcare appointments between **patients, doctors, and receptionists**. It allows users to book, view, and manage appointments efficiently.

## 🚀 Features
- User authentication
- Role-based access (Patient, Doctor, Receptionist)
- Appointment booking and management
- Doctor availability tracking
- Responsive UI

## 🛠️ Technologies Used
- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn-ui
- Spring Boot
- MySQL

## ⚙️ Installation

```bash
git clone <YOUR_GIT_REPOSITORY_URL>
cd Healthcare-Appointment-Management-System
npm install
npm run dev
```

Open in browser:

```
http://localhost:5173
```
